# 下载安装：C/C++ Project Generator 扩展

下载安装完成后，按 ctrl+shift+p 打开命令面板，输入 create C++ project，按回车后可以选择保存工程的文件夹

创建好会后生成几个目录：

.vscode：里面放一些配置文件之类的，如 launch.json、setting.json、tasks.json

include：存放头文件

lib：存放项目所依赖的外部库文件

output：存放生成的可执行文件

src：存放源码文件

还有个 Makefile 文件

# SEO 是什么，每天的工作内容

# 什么是 SEO

  • 网站排名

  • 搜索引擎优化

    • 搜索引擎

      360、百度、谷歌

    • 优化

      用户搜索某关键词时，百度会推荐上千万条结果，但用户只会点击眼下首页的几个答案。如果你的产品想在互联网上推广，提升自己网站的曝光度，就要用到 SEO，把排名排上来

  • Search Engine Optimization

# SEO 工作内容

  • 网站更新

    更新文章，发 1~2 篇新文章。写文章也有技巧，是运用到 SEO 的

  • 数据分析

    • 网站后台数据

      调查网站后台数据，根据数据变化来做调整

    • 竞争对手分析

      比如同行的排名比你高，就要调查他到底哪里做得比你优秀。我们只需要在这一点超过他们，或在他不足的地方做得更好，我们的网站排名就会比他更靠前

  • 网站的页面价值

    对自己或他人的网站进行估价

  • 后期规划

    • 前期

      网站刚刚搭建就属于前期，没有用户，只有百度来检查你网站数据，网站数据全部来自百度

    • 后期

      网站运营很久之后，有一定的原始用户积累，这时网站想怎么改都行。数据全部来自用户

  • 网站推广

    比如链接推广或发外链

  • 全搜索覆盖

    不管做淘的引流还是自媒体（软文的推广）都属于 SEO。SEO 是一种思维方式，不单只是网站排名。当你成为一个合格的 SEO 的时候，就几乎能运营一个个人 IP

# 认识百度算法规则

  • 百度的排名机制

    由一个机器来控制，它会对网站进行评估，通过算法规则，从多方面计算网站得出分数，网站得分越高排名就越高

  • 百度的两种得分计算方法

    • 单页面

      对一个页面进行计算。这个页面有排名，但其它页面不一定有排名。只能或得这一个页面的用户成交量

    • 整个网站

      把所有单页面整合起来。整个网站得分会影响之后发布的文章的排名速度（排名很容易，但速度不一定能够解决）和收录速度、排名的几率和收录的几率、之后网站的权重值

# 网站排名的流程数据

SEO 就是优化该流程里面的每一项指标数据

  1. 放出蜘蛛

    百度首先要拥有你页面的数据，数据来自于蜘蛛的抓取。蜘蛛只是一种叫法，它事实上是一串代码，谷歌的叫机器人。百度会放出一串代码来到你的网站

  2. 蜘蛛来访抓取

    抓取之后百度把你的网站初步进行了一波计算识别，然后它把能抓取的数据都记录下来

  3. 页面的收录

    百度把你页面的数据放在自己的数据库。只有放在数据库的页面才会在百度上或得排名展示的机会

  4. 算法计算得分

    页面下载好之后，百度就会对它进行一些计算

  5. 排名

    得分越高排名就越高

# 正确认识搜索引擎爬虫

# 什么是爬虫

百度的服务器里面会发出一串数据，到整个互联网上面的各个页面进行下载，那这串数据就是爬虫

📗各大搜索引擎都有自己的爬虫：百度蜘蛛、360 蜘蛛、谷歌机器人

# 模拟普通用户

如果某网站不能帮助使用百度的人、不能解决用户需求，百度就不会给你排名。模拟普通用户就是初步检查网站的一些基本信息。好比凡是需要 VIP 才能访问的，或者需要登录才能访问的页面，百度都不会抓取，直接就离开了。

📗比如说 QQ 空间需要登录才能访问，这种页面百度就不会执行页面的收录

📓补充：百度蜘蛛来网站的时候，有两个步骤 —— 来访、下载之后带入数据返回。百度需要计算的页面有成千上亿个，这些页面都需要一一放出蜘蛛去爬行，爬行之后蜘蛛就会带页面数据返回百度，返回百度的过程中就会进行初步的计算了。这个时候如果你页面数据不合格，下一次蜘蛛的爬行次数就会减少，因为百度知道这个页面没有存在价值，就直接卡死在第一个步骤放出蜘蛛

# 网络爬虫的管理规则

  1. 链接长度

    在放入蜘蛛返回的时候百度会初步计算，它不喜欢某站点时，之后的爬行几率就会比较少，爬虫数量会降低

    • 中文链接

      www.一叶.com www.一叶.com/老师

      蜘蛛识别难度会增加很大

    • 英文链接

      www.yiye.com

    链接为爬虫的通道

  2. 网站识别回访

    蜘蛛来访页面后会带一些数据返回百度：

    • 网站打开速度

      打开速度越慢，蜘蛛越没耐心等待（蜘蛛模拟普通用户，相当于模拟真实的用户在等待）

    • 识别难度

      有些东西蜘蛛不识别，或识别难度很大。比如图片识别难度很大，视频完全不识别或识别难度极大

  3. 有效收录

    有价值的页面

  4. 爬虫越多，越容易被收录

    某数据第一只爬虫识别不了，可能后面来的爬虫就能识别，所以爬虫越多，越容易识别出网站内容

    想要网站蜘蛛更多的做法：一个页面起码有 20-30 个链接。蜘蛛访问页面之后不会离开，它会接着下一个页面继续访问，那么就增加了蜘蛛的存留率

  5. 访问日志

    只要有访客来到网站，服务器就会自动生成一个文档，记录当天数据值。我们可以通过这个数据值，观看当天蜘蛛的访问情况

# 收录是什么？如何快速提升网站的收录数量

# 收录是什么

百度放出蜘蛛，蜘蛛来到页面，了解页面基本数据之后带回给百度。这时百度就会经过一番计算判断这个页面有没有存在价值，有没有必要展现给来搜索的用户，页面有价值就会被收录起来。只有被百度收录起来的页面才会有排名的机会

百度会对页面的收录评等级：

  • 有效收录
  • 无效收录

# 收录的高质量计算规则

# 收录率

收录的数量 / 页面的数量

收录率越大，网站质量越高（不绝对）

# ⚠️不是收录越多，排名的机会越大

📗A 网站的收录页面远多于 B 网站，但 B 网站的访客数量远大于 A 网站

所以做 SEO 拼质量，打造质量就是打造有效收录。如果只追求收录的数量，就是在拼概率

# 有效收录

只有能排名的才叫有效收录

能够给用户解决需求的页面越多，占比越高，这个网站质量就越好

权重 6百度预计流量 10000~49999权重 7百度预计流量 50000~199999
权重 8百度预计流量 200000~999999权重 9百度预计流量 1000000 以上

# 百度运营排名的核心思维

百度所有算法更新都是建立在用户基础上的

❓百度是什么

  • 搜索引擎

  • 公司

    • 赚钱

      百分之六七十的收入来自于竞价（SEM，竞价广告位，有盈利性质）谁给的价格越高，谁的排名越靠前。但是在这里面很多人是烧钱的。烧钱却还是愿意在百度做竞价是因为百度的流量多，百度占据整个搜索引擎市场 70%-80%

    • SEO

      百度只搞竞价不搞 SEO 是运营不下去的

      📗比如说当有人在百度搜索 “手机飞行模式怎么开” 的时候，没人花钱做这个排名。这种排名压根就没有一点盈利性质，这时百度只能通过 SEO 把最能满足用户需求的网站排名上来

      当用户使用百度得到自己满意的答案之后对百度就有一个好印象。如果百度推荐答案是用户不喜欢的，用户就会换下一家的搜索引擎。这也是百度占据 70% 市场的原因，因为它推荐的答案很符合用户需求

# 高级 SEO 的核心数据计算

1. 展现量

比如某排名第九的网站，用户搜索且不滑动的情况下是没有展现量的。但是滑动到下面来看到这个网站，那么就算是一次展现量

展现量越高就说明这个网站排名肯定好

2. 点击量

一个用户点击 n 次，就算 n 次的点击量

点击率 = 点击次数 / 展示量

3. 访问量

访问的客户人数，也就是说你真正获得的流量，你的订单就是在访客量中出的

4. 停留时间

代表网站是否真正让用户喜欢

跳出率 = 来之后立马离开的访客数量 / 总访客数量

# 关键词选择的 4 大技巧

一个前期的站点，也就是刚刚起步的站点，只能够做相应的关键词的排名。选择相应的难度，太难了做不上去，太容易了做上去没有流量

❓怎样去选择相应的难度

✅有以下四个标准，共同作为难度选择的参考，而不是单凭某一个标准

  1. 指数

    百度有个平台 “百度指数”，它统计了所有使用百度人群的搜索习惯，我们可以在这里知道用户的搜索行为

    指数越高越难做排名。因为用户搜索得多，流量就越多，大家都想排名，竞争大

    对于一个新站，可以选择两个指数为 500 左右的关键词，作为我们网站前期的目标

  2. 相关结果

    百度在某个关键词排名上面，“相关结果约 71,800,000 个”，即有 7000 多万的页面想跟你竞争这十个位置的自然排名

    新站选择 1000 万，一般 1~2 个月就给排名了

  3. 广告位

    新站选择 3 个

    一个页面最多只展示 5 个广告位，不是只有 5 个人做排名，而是说几十个人在这上面花钱，但百度只展示花钱最厉害的

    广告位在 3 个左右的时候，说明愿意在这个关键词上花钱的人数还可以，有商业价值（只有有商业价值的广告位，人们才会去花钱做广告位），但也不算是太难

  4. 首页数量

    10 个自然排名中，点进去即网站首页的链接数量。把首页放在某个关键词下排名，说明人家很重视这个关键词排名

    新站选择 4 个

# 网站 TDK 正确写法

# TDK 是什么

  • T itle：网站标题
  • D escription：网站描述
  • K eywords：网站关键词

# 对于网站的重要性

如果 TDK 里没有你要排名的关键词，那么你的网站几乎不会有排名

一个网页 40% 的得分都来自于 TDK

# 网站标题的书写规格

关键词 1 - 关键词 2 - 关键词 3 - 品牌词

或 3 个关键词组成一句话

32 个字左右

# 网站描述的书写规格

描述是吸引用户点击的

  • 规格

    • 字数

      60-80。若超过，则不展示，以省略号代替超出内容

    • 抓住用户需求去写描述

      1. 用户的痛点

        可以通过下拉、相关搜索得知用户痛点

        "空压机是干什么用的" 不需要解决，因为问 “是干什么” 的用户不会购买空压机。我们只需要解决我们的意向用户的需求 —— 品牌的需求、价格的需求、维修的需求、操作方法的需求

      2. 行业优势

      3. 包含关键词

  • 举例

    xxx 公司 13 年行业研发（行业优势），专注打造高质量空压机品牌（品牌的需求），主营：空气压缩机、高压空压机、无油空压机（包含关键词）等设备研发，安全、耐用保修（维修的需求）

    ⬆️字数也达标

# 网站高阶栏目打造

点击分为：

  • 站外点击

    用户被 TDK 吸引来

  • 站内点击

    栏目的作用：引导用户点击

正确的栏目规划：

  • 数量

    栏目数量一般不超过 9 个，6 个差不多

    📗假如一个网站中有一个页面得分 100 分，排在第 1 位，这时有一定用户基础，这些用户可以带动网站里其他页面的排名。如果有 10 个栏目，每个栏目最多得到 10 分，如果只有 2 个栏目，每个栏目可得 50 分。栏目数越少，栏目里面的文章就越容易排名

  • 抓住用户需求

    栏目可划分为：

    野生黑枸杞：产品的特点

    黑枸杞的颜色：因为黑枸杞的真假可通过泡水的颜色来辨别，所以它和黑枸杞的真假实际上是一种需求。但是有一些用户了解黑枸杞的特性，有些不了解，了解的人搜索颜色就行，不了解的人看着黑枸杞的颜色不懂

  • 栏目的顺序

    越重要的栏目位置越靠前

# 高质量锚文本解决网站不收录

  • 锚文本是什么

    点击就能跳转的文字

  • 锚文本的作用

    1. 增加网站的点击率（这里通过增加站内点击）

    2. 增加用户停留时间

    3. 增加蜘蛛的爬行（存留）

      因为每一个链接都是蜘蛛的入口。当蜘蛛来到网站进行爬行的时候，从上面爬到下面来的时候，突然碰到一个链接（锚文本），就会爬到链接的网页去，又在那个网页给他提供一个链接，它又会爬到这个链接去。这个时候它在网站里像个迷宫一样反复地爬，不会出去，那么网站的蜘蛛存留率就会很高，会带走网站很多的数据。整个网站的收录率会大幅提高

  • 高质量锚文本的书写要求

    1. 文字与链接内容匹配

    2. 文字最好为跳转到的页面的要排名的关键词

    3. 锚文本的设置密度不能超过 1%

      📗1000 字的文章设置 4 个就够了

    4. 不能多个链接指向一篇文章

      会造成大量回链

    5. 站在用户需求（百度监测）

      某文章内的锚文本没有任何人点击过，那么这种锚文本无价值，没有给用户提供任何有兴趣的价值。网站内如果有大量没人点击过的链接，就会拖垮你这篇文章的点击率，百度就会认为网站很差，导致网站很难排名

    举例

    写一篇文章《SEO 是什么》，站在用户角度帮他解决需求，搜索人群肯定是 SEO 小白。那在文章里出现 “外链” 或 “内链” 这种关键词的时候，用户肯定不懂关键词的意思，我们就设置锚文本解释一下内链的作用和外链的作用是什么，这种链接用户肯定会点击，这就是高质量的锚文本

# 网站的文章的书写技巧

如果用户来网站之后点击锚文本，发现这篇文章他不是很喜欢，立马离开了，反而会增加网站的跳出率，那么网站这篇文章就很难获得排名

  1. 原创文章能否增加排名几率

    • 百度鼓励原创，追求用户体验

    • 如果追求更新速度或不会写文章，可以伪原创，使用 PP文章伪原创洗稿降重助手3.3(1).exe

  2. 文章的写作技巧

    1. 图文结合

      1~3 张图

    2. 字数要求

      800 左右

    3. 不能乱写

      涉及有效收录。如果用户不喜欢，用户就不愿意停留、点击、浏览；就算用户喜欢看，但不一定搜索得到

      要根据关键词来写，要挖掘大量的长尾关键词

# 高级 SEO 外链算法

# 外链是什么

别人的网站上放置自己的链接

站外链

📓网站内部的链接就叫内链，比如锚文本

# 外链的作用

  1. 吸引百度蜘蛛和用户

  2. 文章的引用法则

    该文章被其他网站引用越多，文章质量越高。只要多发文章多发外链，排名就能做上去

  3. 百度算法的出现：绿萝算法

    打击外链

    对外链进行检测，看外链是否是真正的优质外链

# 网站 SEO 外链发布技巧

  1. 发布的形式

    • 文字

      把你的页面地址在别人的网站以文本形式打出来

    • 链接（锚文本）

    • 图片

  2. 发布的技巧

    • 选择高质量外链平台
    • 对链接有文字描述（引导点击）
    • 内容与实际相符
    • 一段文字不要发布多个平台（否则会被绿萝算法打击。一条高质量的外链可能抵得过几千条垃圾外链）

📗在知乎回答问题，附上自己的网页地址并进行描述

# 利用友情链接带动网站数据

  • 友情链接是什么

    双方互通的外链。你网站蜘蛛到我网站来，我网站的蜘蛛到你网站来，可以带动网站之间的蜘蛛的互动性。比如我有一个新站，百度很少给我发送蜘蛛，很少用蜘蛛来爬我的网站。这时交个友链就可以叫他网站的蜘蛛来到我网站，这时百度会把我网站的信息带到百度数据库里面，从而带动我网站的收录和权重

  • 友情链接的交换规则

    1. 30 个左右

      选择周边行业（比如做网络营销的，SEO 就是属于网络营销的一环）或同行业（做 SEO 的网站）

    2. 对方权重高

    3. 选择优质站点或站内优化做得好的潜力股

    4. nofollow

      禁止蜘蛛对该链接分配权重、禁止蜘蛛爬行该链接。你网站的蜘蛛可以到我这里来，但我这里的不会到你那边去

    5. 内页

      同等优化程度的前提下，别人的内页和你的首页换，那么你会比较亏，因为首页的蜘蛛比内页多

# 闪电算法：秒排的数据要求

  • 秒排是什么

    网站文章或页面一被收入就立马获得排名，甚至排名进首页

  • 闪电算法是什么

    百度针对网站打开速度提出的一项优化规则

    • 大于 3 秒：网站会被减分或很难获得排名
    • 2~3 秒：百度给你默认排名方式，不加分也不减分
    • 小于 2 秒：加分项
    • 小于 1 秒：秒排的必要条件

  • 影响网站数据的相关文件

    1. 模板

      要选择 css 和 js 比较少的模板

    2. 图片

    3. 服务器

      复制域名，粘贴到站长工具的网站测速，查看分析，就会模拟各个地区的用户加载时间

  • 怎么针对性调整网页

    打开一个网页 -> 检查 -> Network -> 加载时长大于等于 1s 的文件就不合格 -> 右键 -> Open in Sources panel -> 能看到到底是什么文件不合格 -> 优化或压缩一下再上传

# 为什么学 SEO 不先搭建网站

第一印象规则

在百度放出蜘蛛第一次接触到你网站的时候会有个评估。第一种情况：它接触到你这个网站的时候，发现各项数据都优化得很好、很符合用户体验，它就知道站长会做 SEO、会做用户体验，就会认为你之后发布的文章和页面也是具有用户体验的，从而加大排名力度。第二种情况：百度放出的蜘蛛接触到你网站的时候发现爬行、识别起来很难，没什么用户体验，站长显然不会做 SEO，百度之后对你的评估程度会更严厉

这就是为什么一些 SEO 做得好的站长选择把网页优化好之后再上线进行一个排名

# 选择一个复合 SEO 的网站

  • 域名是什么

    www.baidu.com

    www.bilibili.com

    域名一旦选择好，直接决定了网站之后一个长久运营以及用户对网站的记忆属性

  • 域名的组成

    • 前缀：www
    • 主体：baidu、bilibili
    • 后缀： .com 、 .net 、 .cn

  • 域名的选择

    • 主体

      • 选择你产品的英文、拼音

      • 品牌（公司、个人 ip）、优化关键词（产品词）

        📗五金建材： www.wjjc.com 、 www.jiancai.com

# 在别人的平台下建站：域名权重的划分

  • 域名等级划分

    • 一级域名

      baidu.com

    • 二级域名

      www.baidu.com

      ziyuan.baidu.com

    • 三级域名

      bbs.ziyuan.baidu.com

  • 网站的排名有利程度

    • 主域名（一级域名）> 二级域名 > 栏目（www.huabaike.com/hhzl/） > 内页（www.huabaike.com/yzfh/33035.html）

    • 链接越长越难排名

# 成交率最高的网站布局

网站排版

排版是在我们搭建网站之前，根据调查出来的用户数据来做的一些布置

F 型布局

越重要的放置离左上角越近。因为越靠近左上角，用户的目光以及点击次数会越高

# H 标签的细节优化

  • H 标签的作用

    百度放出蜘蛛之后来到网站中抓取这篇文章，必须要有个重点。搜索引擎蜘蛛会认为 H 标签是网站中或这篇文章中等级最高的一个代码标签，所以这里一般放置需要排名的关键词

    <h1></h1> 一个页面只能书写一个

  • H 标签的实用技巧

    • 等级划分

      h1~h6

# 网站 nofollow 实用技巧

  • nofollow 的作用

    告诉搜索引擎不要让蜘蛛来爬取某条链接或对这条链接分配分数和权重，为了不浪费权重

  • 应用范围

    1. 外部链接

      • 别人到你的网站上挂外链

      • qq 客服链接

      • 广告

    2. 不重要的页面

      “关于我们”、“联系我们”

# 网站 alt 代码优化

  • ALT 的作用

    向百度给图片做说明

  • ALT 的实用规则

    • 每一张图片都要加

    • 把要排名的关键字放进去（但不能每一张都直接加，否则会被百度判定为故意优化）

      📗比如要排名的关键词是 “小狗”，alt 就赋值 "奔跑的黑色小狗"

# 企业及其他类型网站的 SEO 优化方案

一系列的优化方法都是根据一个点来的：用户的数据分析

做网站要追求：人无我有，人有我优

  • 企业网站的优化方案

    1. 产品类站点
      • 产品分类
      • 热门产品
      • 价格
      • 维修
      • 公司的保证
    2. 服务类站点
      • 客户案例
      • 流程
      • 区域
      • 价格

  • 其他网站的优化方案

    1. 商城内站点的优化方案

      • 图片很多

        要高清图片，但大小不能太大，或搭建网站时要选择预加载

      • 种类特别多

        种类要做划分，一个行业做得越垂直（也就是越专业，比如专业做拉面大于兰州菜馆），这个网站往往就越容易排名

      • 不追求排名，追求转换

    2. 资讯类站点的优化方向

      • 细分

        比如花可以分为花的种植、花的种类、花的养护方法等

    3. 小说类站点的优化方向

      • 文章内容无法改变，因为文章都是采集的、复制粘贴来的
      • 小说的介绍、简介
      • 作者的介绍
      • 排行榜

    4. 视频类网站的优化方向

      • 视频内容无法改变
      • 电影的打开速度、加载速度
      • 排行榜

    5. 个人博客

      • 个人了解这个行业，然后去做知识分享
      • 一般不追求排名

# 怎样快速学好 SEO

  • 学好
    • 全面性。任何行业的网站都可以上手
    • 能自主应对百度算法的调整，甚至网站有排名之后百度算法更新对你没有影响
    • 换位思考
  • 如何快速学习
    • 不闭门造车，学会与人沟通
    • 讨教前辈，节省时间换取机会

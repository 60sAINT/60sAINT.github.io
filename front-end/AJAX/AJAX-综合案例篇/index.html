# Bootstrap 弹框

Bootstrap 弹框 = Modal = 模态

功能：不离开当前页面，显示单独内容，供用户操作

# 通过属性控制

步骤：

  1. 引入 bootstrap.css

  2. 准备弹框标签，确认结构

  3. 通过 bootstrap 内部的自定义属性，控制弹框的显示和隐藏
